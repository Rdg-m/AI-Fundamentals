# Módulo 1: Introdução ao Machine Learning

## Visão geral do módulo

### Contexto
Este módulo estabelece a fundação conceitual para todo o curso. Ele desmistifica termos como Inteligência Artificial, Machine Learning e Deep Learning, explicando suas diferenças e relações. Apresenta os três paradigmas principais de aprendizado (supervisionado, não supervisionado e por reforço) e introduz a notação matemática fundamental que será utilizada ao longo dos módulos seguintes. **Sem dominar esses conceitos, será difícil compreender os algoritmos específicos que virão nos Módulos 2, 3 e 4.**

### Estrutura
Este módulo contém **3 notebooks**:

1. **01_fundamentos_ia_ml.ipynb** — Fundamentos de IA e Machine Learning
   - Diferenciação entre IA, ML e Deep Learning
   - Subáreas da Inteligência Artificial
   - Aplicações práticas no dia a dia

2. **02_tipos_aprendizado.ipynb** — Tipos de Aprendizado de Máquina
   - Aprendizado Supervisionado (com rótulos)
   - Aprendizado Não Supervisionado (descoberta de padrões)
   - Aprendizado por Reforço (agentes e recompensas)

3. **03_matematica_ml.ipynb** — Conceitos Matemáticos Fundamentais
   - Dados, características e vetores de características
   - Espaço de características ($\mathbb{R}^d$)
   - Notação matemática básica para ML

### Tempo Estimado
- **Notebook 1:** 20-30 minutos (leitura conceitual)
- **Notebook 2:** 25-35 minutos (leitura conceitual + reflexão)
- **Notebook 3:** 30-45 minutos (conceitos + notação matemática)
- **Total do Módulo:** 1h15min - 1h50min
